# Antennas and Propagation
The report documents as well as the code comments are all in greek.

These assignments cover antenna design and analysis: radiation patterns, gain, directivity, frequency response, impedance, etc. The types of antennas analyzed are mostly simple dipoles and linear arrays, but biconical and helical antennas are examined as well.

The solutions involve both theoretical (pen-and-paper math) and computational (MATLAB, 4nec2) methods.
